# MFM NCR24 Church Website

A simple interactive church website for Mountain of Fire and Miracles Ministries NCR24 chapter.

## Features

- Home page with hero section
- About section
- Services listing
- Announcements display
- Contact form
- Admin panel for managing announcements

## Admin Access

Go to `admin/index.html` and login with password: `admin123`

## Setup

Open `index.html` in a browser to view the site.

For local development, you can use a local server like Live Server extension in VS Code.

## Note

This is a demo version using localStorage for data persistence. For a production site, implement a proper backend database.
